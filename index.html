Aries Agent

Give autonomous agents a place to run with real boundaries: the host keeps identity and secrets, every tool grant is scoped, and each run leaves a clear trail.

Run Aries from source
  1. Trust Host-held trust

    Identity, secrets, policy, routing, and approvals stay with the host.

  2. Scope Narrow by default

    Each runner receives only the tools, skills, MCP servers, and mounts it needs.

  3. Proof Clear run history

    Connectors, schedules, and agent calls leave receipts you can inspect.

Quickstart

Start from source.

git clone https://github.com/haozhongsheng/aries.git
cd aries
make all && aries
